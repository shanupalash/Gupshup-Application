💬 Gupshup Application

Vercel Link : https://gupshup-app-phi.vercel.app/login


Gupshup is a real-time social networking application designed to enable users to share posts, join communities, and interact with others in a seamless and engaging way. Think of it as your personal digital café — where conversations never stop.
🚀 Key Features

    📝 Post Creation: Users can create text/image posts and share them publicly or within a community.

    💬 Comment & Like: Engage with content through comments and reactions.

    👥 User Authentication: Secure sign-up and login using JWT.

    🌐 Communities: Create or join interest-based communities.

    🔎 Search & Discover: Find posts, users, and communities easily.

    👮 Moderation Tools: Admin/moderator features for managing communities.

    🛡️ Role-Based Access Control: Admins, moderators, and users have different permissions.
